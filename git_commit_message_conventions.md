- Format of the commit message
```
<type>(<scope>): <subject>

<body>

<footer>
```

- Allowed <type>  
\# This describes the kind of change that this commit is providing.
```
feat (feature): 新功能
fix (bug fix)
docs (documentation)
style (formatting, missing semi colons, …): 格式, 不影响代码运行的变动
refactor: 重构
test (when adding missing tests)
chore (maintain): 构建过程或辅助工具的变动
```
